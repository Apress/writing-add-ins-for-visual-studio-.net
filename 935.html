Errata for Writing Add-ins for Visual Studio .NET.

 

1. The definition for the OnConnection Method on page 28 should read:

 

   Public Sub OnConnection(ByVal application As Object, _

      ByVal connectMode As Extensibility.ext_ConnectMode, _

      ByVal addInInst As Object, ByRef custom As System.Array) _

      Implements Extensibility.IDTExtensibility2.OnConnection

 

2. On page 23, the value for LoadBehavior should be 1 and not 0.

 

3. On page 35, the registry key for listing an add-in with the Add-in manager is given.  This registry setting is for Visual Studio 2001.  For Visual Studio 2003, the registry key should be changed in one character.  The "7.0" must be changed to "7.1" in order for the Add-in manager in VS 2003 to find the values to connect the add-in.  An add-in compiled (DLL) under VS 2001 will work without recompiling if the aforementioned registry adjustment is made.

 

 

4. Chapter 6, Page 165, Line 11 should read ' s = "btnTestButton"', instead of ' s = "txtTestButton"'
